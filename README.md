## Directory Structure
```tex
barm
 ├── barm-archetype 微服务节点maven 脚手架
 ├── barm-common    项目公用包
 ├── barm-template  微服务节点模板项目
 ├── barm-tree      分布式ID服务
 └── README.md
```
